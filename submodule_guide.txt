adding new submodule

  git submodule add https/link/to/submodule path/location
  git commit
  git push

updating submodule
 
  go into the submodule and do git add, commit, push 

  then go into the parent module 

  git add path/location
  git commit
  git push

when pulling 

	git pull

	git submodule update --init --recursive 
		if new submodule
	git submodule update --remote --merge
		if submodule already exist
